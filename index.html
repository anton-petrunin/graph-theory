Extra pearls in graph theory

I used these topics together with the book "Pearls in graph theory" to teach an undergraduate course in graph theory at the Pennsylvania State University. I tried to keep clarity and simplicity on the same level.

You can downloade it here "Extra pearls in graph theory".

If you would like to contribute, you will have to use Git and LaTeX. The following command will create a local copy of the source code for you.

git clone https://github.com/anton-petrunin/graph-theory.git

You can also create an alternate version, see the license below.

Table of contents:


1. Ramsey numbers continued
2. Deletion and contraction
3. Matrix theorem
4. Marriage theorem and its relatives
5. Polynomials
6. Generating functions

Hope that someone will find it useful for something.
